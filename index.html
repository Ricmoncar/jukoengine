How it works: set Apply to → Selection, highlight part of the text, then click a tool to style just that bit. Set it to Entire to change the default for the whole dialogue. Styles nest, and you can type the tags by hand too: [c=#ff0000]red[/c] · [rb]rainbow[/rb] · [a=wave]animated[/a] · [stable]no glitch[/stable]. Leave everything untouched for the classic green matrix thinygy.
